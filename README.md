## Specifications

|  Behavior | Input  | Output  |
|---|---|---|
| Return a scrabble score for a letter | a | 1 |
| Return a scrabble score for a word | epicodus | 13 |
| Return a scrabble score for a sentence | i love coding | 18 |

### Scoring Guide

* A, E, I, O, U, L, N, R, S, T => 1
* D, G => 2
* B, C, M, P => 3
* F, H, V, W, Y => 4
* K => 5
* J, X => 8
* Q, Z => 10
